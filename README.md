# AI Assignment 6 – California Housing Regression

## Project Overview

This project demonstrates a complete machine learning workflow using the California Housing Dataset from scikit-learn. A Linear Regression model is developed to predict median house values based on housing features.

## Objectives

- Import required Python libraries
- Load the California Housing Dataset
- Explore and understand the dataset
- Check for missing values
- Visualize feature relationships using scatter plots
- Train a Linear Regression model
- Evaluate the model using RMSE and R²
- Predict house values for new input data

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Model Performance

- RMSE: **0.7456**
- R² Score: **0.5758**
